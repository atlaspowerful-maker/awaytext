# Projet : SMS Auto-Responder

## Concept
Micro-SaaS à 2€/mois pour artisans et professionnels.
Répond automatiquement aux SMS reçus pendant les congés/vacances.

## Problème résolu
Les pros (artisans, commerçants, indépendants) reçoivent des SMS clients
même en vacances. Pas de solution simple pour auto-répondre.

## Solution
- Forward SMS vers un numéro de secours OU
- Réponse automatique personnalisée
- Activation/désactivation simple

## Stack envisagée
- Backend: Node.js + Express
- SMS API: Twilio ou Vonage
- DB: PostgreSQL ou SQLite
- Frontend: Simple HTML/React

## Pricing
- Gratuit: 10 SMS/mois
- Premium: 2€/mois illimité
